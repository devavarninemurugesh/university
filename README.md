How to Run Locally

Clone the repository:

bash   git clone https://github.com/devavarninemurugesh/meridian-university.git

Open index.html in any browser — no setup needed!

Deployment on GitHub Pages

Push index.html to your GitHub repository
Go to Settings → Pages
Set source to Deploy from a branch → main → / (root)
Click Save
Your site will be live at:

   https://devavarninemurugesh.github.io/meridian-university/
